第三讲：类和对象
<!--more-->
一、 课程回顾
面向对象基本概念

常引用 (const T &)：指向常量的引用，不能通过它修改原变量。常用于函数参数，既能避免拷贝开销，又能保证实参不被修改。  

```cpp
const int &r1 = 10;      // 正确，引用常量
const int &r2 = a + b;   // 正确，引用表达式结果
int &r3 = 10;      //报错！！   // 普通引用不能引用常量
```

访问权限：  

public：任何地方都可访问。  

private：仅类内部可访问（推荐数据成员设为private）。  

protected：类内部和派生类可访问。  

函数重载：同一作用域内，函数名相同但参数列表（个数或类型）不同。  


特殊函数
构造函数：对象创建时自动调用，初始化对象。  

复制构造函数：用一个对象初始化另一个同类对象时或传值/返回时调用。  
```cpp
class Student {
public:
    // 构造函数：对象创建时自动调用
    Student(string name) {
        m_name = name;
        cout << "构造函数：学生 " << m_name << " 已创建" << endl;
    }

    // 析构函数：对象销毁时自动调用
    ~Student() {
        cout << "析构函数：学生 " << m_name << " 已销毁" << endl;
    }

    void sayHello() {
        cout << "我是 " << m_name << endl;
    }

private:
    string m_name;
};
```
调用时机：
初始化对象：A a2(a1); 或 A a2 = a1;
对象作为函数参数（值传递）。
对象作为函数返回值（值返回）。



析构函数：对象消亡时自动调用，用于清理资源。



this 指针

在非静态成员函数内部，this 是一个指向当前对象本身的指针。

它的作用就是让成员函数知道当前是哪个对象在调用它。例如，return *this; 可以返回对象自身。

二、 新知识点精讲
内联函数

作用：通过在函数返回值前加 inline 关键字，建议编译器在调用处将函数体代码直接展开，以消除函数调用的开销。

适用：函数体短小、频繁调用的函数。


内联成员函数：

在类内直接定义函数体（如 void func() { }）会隐式成为内联函数。

在类内声明，类外定义时加上 inline 关键字。

成员函数重载与参数缺省

参数缺省：可以为函数参数指定默认值。

规则：缺省参数必须从右向左连续提供。调用时，只能省略最右边的连续参数。
二义性：避免同时使用缺省和函数重载导致调用不明确。



静态成员 (static)

静态成员变量：所有对象共享一份数据，不属于任何特定对象。
静态成员也可以通过对象、指针、引用进行访问。

必须在类外单独定义和初始化，如 
```cpp
int CRectangle::nTotalNumber = 0;//初始化在类外，不能再加static

//调用
CRectangle r;
CRectangle* p = &r;
CRectangle& ref = r;

r.PrintTotal();      // 通过对象调用
p->PrintTotal();     // 通过指针调用
ref.PrintTotal();    // 通过引用调用
```

sizeof 运算符不计算静态成员变量的大小。

静态成员函数：

不作用于具体对象，可以通过类名直接调用（类名::函数名）。

不能访问非静态成员变量，也不能调用非静态成员函数。

内部不能使用 this 指针。

常量成员函数 (const)

定义：在成员函数的参数列表后加 const 关键字。
使用：
```cpp
class CTest {
    int n;
public:
    int GetValue() const { return n; }      // 常量版本，const在参数列表后，事实上修饰了隐含的this指针。
    int GetValue() { return 2 * n; }        // 非常量版本
};//根据对象是否是常量来决定调用哪一个，实现重载。

const CTest obj1;
CTest obj2;
cout << obj1.GetValue() << ", " << obj2.GetValue(); // 输出: 1, 2

```

```cpp
void GetValue() const;
```
约束：

不能修改类中的非静态成员变量。

不能调用同类的非 const 成员函数（静态成员函数除外）。

常量对象：

用 const 修饰的对象（如 const Sample obj;）。

只能调用其常量成员函数、构造函数和析构函数。

mutable：成员变量用 mutable 修饰后，允许在 const 成员函数中被修改。


成员对象与封闭类

定义：如果一个类的成员变量是另一个类的对象，这个类就是封闭类。

初始化：封闭类必须通过构造函数初始化列表来显式初始化其所有的成员对象。

```cpp
CCar::CCar(...): price(p), tyre(tr, tw), engine(eNum) { ... }
```
构造与析构顺序：

构造：先构造成员对象（按它们在类中声明的顺序，和初始化列表顺序无关），再执行封闭类的构造函数体。

析构：先执行封闭类的析构函数体，再析构成员对象（与构造顺序相反）。

友元 (friend)

作用：允许一个外部函数或另一个类访问本类的私有成员，打破了封装性，应谨慎使用。

友元函数：一个普通函数或另一个类的成员函数。

```cpp
friend void func(); // 声明一个外部函数为友元
friend void OtherClass::method(); // 声明其他类的成员函数为友元
```
友元类：如果A是B的友元类，则A的所有成员函数都可以访问B的私有成员
。
```cpp
friend class CDriver; // 声明CDriver为友元类
```
特点：友元关系不能传递（A是B的友元，B是C的友元，A不是C的友元，B也不是A的友元！），也不能继承。


    
    
    
补充——————————————————————————————  
1. const char* p    (同char const* p) 
const 在 * 左边，修饰的是指针所指的内容。  
  
p 是一个指向常量字符的指针。  

你不能通过 p 修改它所指向的字符但 p 可指向别的地址。  

```cpp
const char* p = "hello";
// *p = 'H'; // 错误，不能修改内容
p = "world"; // 正确，可以改变指向
```


2. char* const r
const 在 * 右边，修饰指针本身。  

r 是一个常量指针，指向字符。  

你不能改变 r 的指向，但可以通过它修改所指内容。  
```
cpp
char str[] = "hello";
char* const r = str;   // 必须初始化
*r = 'H';              // 正确，修改内容
// r = nullptr;        // 错误，不能改变指向
```  

1. const char* const s
两个 const，一个在 * 左边，一个在右边。  
  
指针本身是常量，且指向的内容也是常量。   

cpp
const char* const s = "hello";
// s = "world"; // false,指针不能改
// *s = 'H';    // false,内容不能改
快速记忆法
const 在 * 左边 → 修饰内容（const 与类型一起理解为“常量类型”）。
const 在 * 右边 → 修饰指针本身。
