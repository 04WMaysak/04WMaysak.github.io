# 机器学习
学习：系统从过程中学习经验。     
机器学习：不需要显示编程让计算机自动学习    
日常应用：机器翻译，交通预测，语音识别等等  

程序：输入数据，输入指令，输出结果。  
机器学习：输入数据，输入答案，得到模型/算法。  
###  分类
###### 模型：
判别式模型：通过学习输入 x 与标签 y 之间的条件概率 p(y|x)，直接预测给定数据所属的类别或标签。     
描述式模型：通过学习输入数据 x 自身的概率分布 p(x) ，来描述数据的统计特征或发现其内在结构。     
生成式模型：通过学习隐变量 z 与数据 x 的关系 p(x|z) ，具备从隐藏特征中生成/重构新数据的能力。  

###### 学习：
监督学习：训练数据带有已知标签  
无监督学习:数据不带标签的      
强化学习：智能体与环境交互，得到数据   

###### 本课程我们主要关注supervised learning和discriminative model
##### 判别式模型
模型 (Model): 一个包含参数  的函数  
标签 (Label): 要预测的类别（分类）或数值（回归）  
标签是标准答案和预测结果。    
模型训练/学习 (training/learning): 通过调整模型参数来拟合 (fit) 训练数据 (training data)    
输入→模型→输出（是什么类/是什么值）  

#### 分类与回归 
回归 (regression): 标签 (label) 是连续 (continuous) 值
 比如，预测一个房子的房价  
分类 (classification): 标签 (label) 是离散 (discrete) 值
 比如，预测狗的品种  




### 模型训练过程：training

训练数据：转化成特征向量。  
标签：转化成计算机可以处理的标签（如1是垃圾邮件，-1不是垃圾邮件）  
模型训练：学习函数f，使fxi约等于yi


### 模型评估testing/evaluation
#### 训练误差：训练集上的平均误差
    通过最小化训练误差来训练模型  
    对分类问题，通常用错误率衡量训练误差：分类错误样本/总样本数  

#### 测试误差：在测试集上的平均误差(testerror)
    训练完成后，衡量模型在新数据上的好坏  
    衡量模型泛化能力（generalization,机器学习最重要的概念）  
    训练误差低不代表测试误差也低————过拟合  
    
#### 划分测试集和训练集(training set,test set)
    给全部的数据，按比例（如90%-10%）划分为训练和测试集（严格隔离！否则数据泄露）
    划分：随机排列，按照时间顺序排列等

#### 过拟合：测试误差远大于训练误差
    特殊规律被错误当成了普遍现象。  

#### 欠拟合：训练完成后，训练误差仍然很大
    首先应当保证不要欠拟合   
    连训练样本都没有拟合好  
    训练集中难以做到f(xi)≈yi  

    传统机器学习：取“甜点”（sweet spot），训练误差没有到最小，但是测试误差最小。

### KNN算法（K近邻） 
最简单的机器学习算法  
对于一个测试样本，用训练样本中距离它最近的k个样本中占多数的标签来预测测试样本的标签  

```
有一堆已经分好类的水果（苹果和梨），现在拿来一个新水果，你想知道它是苹果还是梨。
KNN这么做：
找邻居：在已知的水果堆里，找到离这个新水果“距离最近”的 K 个水果。

看标签：看看这 K 个邻居里，是苹果多还是梨多。
（K在训练前确定）
做决策：根据“少数服从多数”的原则，谁多就把新水果归为哪一类。

KNN 分类：K个邻居里哪种颜色多，我就变什么颜色。（少数服从多数）
KNN 回归：K个邻居们平均是多少，我就大概是多少。（数值平均。也可以是加权平均）
```


#### 优点
    不需要距离
    只需要一个距离函数，默认是欧氏距离

#### 缺点
    需要存储所有训练样本  
    测试时要计算测试样本到所有训练样本的距离  
    有时候很难找一个很好的距离函数

##### 维度灾难：

    在超高维空间中，数据点会越来越集中于立方体的表面（距离趋同）。
    距离在高维空间失去意义，不再能很好地衡量远近  
    k-NN不适合高维空间  

对某个测试样本，设 l 为能包含住其所有 k 近邻的超立方体的边长，（假设点在体内均匀分布）则：
$$l^d \approx \frac{k}{n} \to l \approx \sqrt[d]{\frac{k}{n}}$$
 
```
在一个 d 维单位超立方体（边长为 1，总体积也为 1）中，数据点的分布情况。  
变量含义：d：数据的维度（特征的数量）。
n：样本总数。  
k：找的近邻个数。  
l：为了围住这 k 个邻居，划定的超立方体边长。    
k/n：我们想要找出的近邻点占总样本量的比例    
（也可以理解：d维超立方体中固定边长L<1的一个小立方体，随着d增大，所占立方体体积越来越小。为了继续包含K个点，需要扩大L）
```
  
取n=1000，k=10：  
| d | l |
| :--- | :--- |
| 2 | 0.1 |
| 10 | 0.63 |
| 100 | 0.955 |
| 1000 | 0.9954 |

### 参数化模型
##### 非参数化模型：
例如：k-NN
模型不能被有限参数 (parameters) 定义，或不包含参数  
需要保留训练样本，以对测试样本做出预测  

##### 参数化模型：
模型包含可训练参数，通过拟合训练数据来估算模型参数    
训练好模型参数后，可以丢弃训练数据，仅仅依靠模型参数去预测新样本  
可以写为y≈f(x)，f为包含参数的模型。


#### 线性模型：最简单的参数化模型
线性模型(linear model)

  线性模型 (Linear Models)
  `f(x) = w^T x + b`
  输入 `x ∈ R^d`
  参数 `w ∈ R^d, b ∈ R`，分别称为权重 (weight) 和 偏置 (bias)
  输出 `f(x) ∈ R`
  为每个特征维度学习一个权重，反映其重要性


线性回归 (Linear Regression)
 使用线性模型做回归任务
 标签 `y ∈ R`



例如：

预测房价
  输入特征维度：面积（平米）、房龄、楼层、学区等级（1-5）、物业水平（1-5）
  学习到的权重 $$w = [10, -10, -5, 50, 2]^T$$
  学习到的偏置 $$b = 100$$
  输入向量 $$x = [100, 10, 5, 5, 5]^T$$
  线性模型：$$f(x) = w^Tx + b$$
  输出预测：
  $$100 \times 10 + 10 \times (-10) + 5 \times (-5) + 5 \times 50 + 5 \times 2 + 100 = 1085$$
  偏置的作用：提供一个基础预测值，所有特征在其基础上加权累加


#####  如何训练
通过最小化损失函数。
平方损失函数 (squared loss) 
$$L(f(x_i), y_i) = (f(x_i) - y_i)^2$$
例如：
$$f(x_i) = 8$$
$$y_i = 9$$
$$L(f(x_i), y_i) = (8 - 9)^2 = 1$$
