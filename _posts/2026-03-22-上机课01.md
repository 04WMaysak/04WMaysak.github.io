1. 语法  
在成员函数的参数列表后面加上 const 关键字。  

```cpp
class Student {
private:
    int age;
public:
    // 语法：返回值类型 函数名(参数列表) const
    int getAge() const {
        return age;   // ✅ 允许读取
        // age = 10;  // ❌ 不允许修改
    }
};
```
2. 规则  
 不能修改成员变量.  
    
只能调用 const 函数：在 const 函数内，只能调用其他const 成员函数，不能调用普通的成员函数。  

const 对象只能调用 const 函数：常量对象（const Student s;）只能调用这个对象的 const 成员函数。  

mutable:   
如果你确实需要在 const 函数中修改某个变量（例如用于统计调用次数的计数器），可以将该变量声明为 mutable。  
```cpp
class Demo {
private:
    mutable int count; // 即使在 const 函数中也可以修改
public:
    void log() const {
        count++; // ✅ 允许
    }
};
```  
const与非const同名函数构成重载！！！  
调用规则：非 const 对象优先调用非 const 版本；const 对象只能调用 const 版本（非 const 版本无法被const对象调用）。  

补充：奇思妙想：  
非const如何调const版本：  
1，对其使用const引用。  
2，使用const_cast  
const_cast:  
```cpp
const_cast<new_type>(表达式)

//不修改原始对象，返回一个转换后的值。//可能导致问题
```
```cpp
class Array {
    int data[10];
public:
    const int& operator[](int i) const {   // const 版本：只读
        return data[i];
    }
    int& operator[](int i) {               // 非 const 版本：可写
        // 调用 const 版本，再 const_cast 去掉 const
        return const_cast<int&>(const_cast<const Array&>(*this)[i]);
    }
};
```
